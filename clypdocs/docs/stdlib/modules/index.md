This folder documents the small `clyp.std` modules shipped with the
project (thin wrappers around Python functionality).

Available modules:
- `fs` — filesystem helpers
- `json` — JSON helpers
- `math` — math utilities
- `random` — randomness utilities
- `time` — time helpers

See the individual pages for details.
